Descripción

El software de imagen QuickVision está destinado a las consultas dentales e incluye una base de datos de pacientes, un módulo de imagen y un esquema dental. Puede utilizarse con equipos digitales de radiología dental, como un hub que centraliza todas las imágenes del paciente. QuickVision permite a los usuarios llevar a cabo las funciones de imagen habituales en una consulta dental: adquisición y visualización de imágenes, acceso a las herramientas para realzar las imágenes, medir, ampliar, poner anotaciones en la imagen y compartir datos e imágenes.



Configuración mínima

Procesador: Core i3 3 GHz o más
